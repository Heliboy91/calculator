Help

  1. You can use either the buttons or you can use the keyboard to enter values
  2. You can use "Backspace" to delete the last character
  3. Use "C" to erase all the characters
  4. Use "Enter" to run the calculation
  5. If you omit number after an operation sign, like " 3+ ", we assume a 0
  6. If you omit a number before a dot, like " .33 ", we assume a 0
